Query/Redeem Point Pack

1. Note your Pack Code

Note the Pack Code in red circle: 3ED3D658-XXXX-XXXX-XXXX-771D4B5BE461-100 (X is masking the real letters).

2. Find your account ID in app

Note the account ID in red circle: 1000X (⚠️ You should use your own account ID ⚠️)

3. Query the pack

Put the Pack Code from Step 1 in the box below, and click Query button. This step just shows the content of the pack, and will NOT consume the code.

4. Redeem the pack

Leave the previous box unchanged. Copy the Account ID from Step 2 in the box below, and click Redeem button. This step will CONSUME the code, and add the points from the pack to the account.
